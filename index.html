MODALIDAD DE PROGRAMACION Y DISEÑO WEB

Seccion 1006

En esta carrera aprendemos a crear páginas y aplicaciones web desde cero. Hacemos dos cosas principales: El Diseño: Que la página se vea bonita, con buenos colores, textos y que funcione bien en el celular (usando HTML y CSS). La Programación: Hacer que la página funcione de verdad, como crear botones que sirvan, formularios de registro o carritos de compras (usando JavaScript). Es una modalidad muy práctica. Te la pasas inventando proyectos y resolviendo problemas con código. Básicamente, si usas una app o una página web todos los días, nosotros aprendemos a fabricarla."
